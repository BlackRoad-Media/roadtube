# RoadTube Roadmap

## Phase 1: Foundation (Months 1-4)
- [RC] Video upload + transcoding (H.264, adaptive bitrate HLS)
- [RC] Custom video player with chapter support
- [RC] Voice-to-LaTeX engine v1 — 90%+ accuracy on common math
- [RC] LaTeX overlay rendering in player (KaTeX)
- [RC] STEM topic taxonomy — math, physics, CS, engineering, biology, chemistry
- [RC] Creator authentication + basic dashboard
- [RC] PostgreSQL schema — multi-tenant, revenue-aware
- [RC] CDN setup for video delivery
- **Milestone**: Alpha with 10 founding STEM creators

## Phase 2: Creator Economics (Months 5-7)
- [RC] Revenue system — 90%+ creator share, real-time reporting
- [RC] Payment integration — Stripe Connect, weekly payouts
- [RC] Zero transaction fees on free educational content
- [RC] Creator analytics — views, retention, revenue breakdown
- [RC] Chapter auto-generation from transcript
- [RC] Embedded quizzes + practice problems
- [RC] Interactive code blocks (Python, JavaScript)
- [RC] Basic auto-visualization — function plots, graphs
- **Milestone**: Beta launch, revenue system live, 50 creators

## Phase 3: AI Tools (Months 8-10)
- [RC] Auto-visualization v2 — 3D surfaces, matrix transforms, algorithm viz
- [RC] Voice-to-LaTeX v2 — expanded domain coverage (50+ math areas)
- [RC] Interactive code blocks — Rust + Go support
- [RC] Spaced repetition for embedded quizzes
- [RC] Viewer progress tracking across series
- [RC] Playlist + course builder
- [RC] Mobile apps — iOS + Android
- [RC] Accessibility — auto-captions, audio descriptions
- **Milestone**: Public launch, AI tools differentiation clear

## Phase 4: Scale (Months 11-14)
- [RC] Live streaming with real-time LaTeX overlay
- [RC] Collaborative video editing
- [RC] Creator API for programmatic management
- [RC] University/institution accounts
- [RC] LMS integration (LTI 1.3)
- [RC] Community translations system
- [RC] Recommendation engine — topic-based (not engagement-based)
- [RC] Creator Pro tier ($29/mo) with unlimited AI tools
- **Milestone**: 500 creators, 100K monthly viewers

## Phase 5: Growth (Months 15-36)
- [RC] Expand beyond STEM — humanities, languages, arts
- [RC] Creator collaboration marketplace
- [RC] Enterprise tier for corporations
- [RC] Podcast/audio mode
- [RC] Desktop app for offline viewing
- [RC] Video embeds for external sites
- [RC] Regional CDN expansion
- [RC] Research partnerships — learning efficacy studies
- **Milestone**: 5,000 creators, 1M monthly viewers
